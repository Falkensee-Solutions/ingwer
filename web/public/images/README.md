# Bilder

Struktur:

- `hero/` – großflächige Bilder für Hero-Bereiche
- `workshops/` – Eindrücke aus Werkstätten (ohne erkennbare Gesichter)
- `materialien/` – Materialien, Tische, Hände bei der Arbeit
- `produkte/` – Werkstattprodukte (Kunst, Plakate, Texte) – nur mit Freigabe
- `orte/` – Räume, Gotteshäuser, Natur

## Regeln

- **Keine erkennbaren Gesichter** ohne schriftliche Einwilligung.
- Bevorzugt: Hände, Materialien, Räume, Natur, Symbolbilder, Gruppen von hinten,
  unscharfe Aufnahmen.
- Jedes Bild benötigt einen **Alt-Text** bei der Einbindung im Code.
- Keine externen Bildquellen ohne geklärte Nutzungsrechte einbinden.
- Bilder komprimieren (Web-tauglich, idealerweise WebP oder gut komprimiertes JPG/PNG).

Bei Unsicherheit ein Bild lieber weglassen.
