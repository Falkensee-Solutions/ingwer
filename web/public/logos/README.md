# Logos der Partner

Hier werden die offiziellen Logo-Dateien der Partner abgelegt.

Erwartete Dateinamen (siehe `src/data/partner.ts`):

- `forum-dialog.svg`
- `ekbo.svg`
- `erzbistum-berlin.svg`
- `house-of-one.svg`
- optional: `berlin-foerderer.svg`

**Wichtig**:

- Bitte ausschließlich vom Partner freigegebene Originallogos verwenden.
- Keine Logos ungeprüft aus dem Internet laden.
- Keine Fake- oder Platzhalter-Logos erstellen.
- Bevorzugt SVG; falls PNG, dann transparent und mindestens 400 px breit.

Solange keine Datei vorhanden ist, zeigt die `PartnerLogoBar`-Komponente einen neutralen
Textplatzhalter mit dem Partnernamen.
